# vuego

> Mobile UI components with Vue.js 2.0

[2.0 Changes](https://github.com/vuejs/vue/issues/2873)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run all tests
npm test
```
